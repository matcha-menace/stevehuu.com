Tooling

To ensure a smooth development I've created many VFX tools for artists and designers in the team to tune. These are made while coordinating with other programmers to integrate into our custom state machine so the designers can tune how aggressive effects are for each different attacks.

These tools include the following:

Customizable Impact Effects

The effects can be assigned to different states The effects are modular with unique parameters for tuning The tools works together with the custom physics settings connected to each character's attack states. Tools for drawing hurt- and hitboxes dynamically on character.

Character Chroma

This is a separate tool which allows designers to change character's shader values and save them to a file. Allowing for easier art direction and adjustments.
